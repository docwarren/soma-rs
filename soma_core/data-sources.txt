free to download clinical databases of variant actionability in cancer
Several databases offer free downloads of clinical variant actionability in cancer, though some may have restrictions for commercial use. Major databases include CIViC, ClinVar, and CGI, which provide downloadable data for research purposes. OncoKB does not offer bulk downloads of its annotated variants. 
Clinical Interpretation of Variants in Cancer (CIViC)
This is an open-access, community-driven resource for interpreting the clinical significance of cancer-related variants. 
Data availability: CIViC provides nightly and monthly releases of its database, with downloadable files in TSV and VCF format.
License: Content is made freely available for public use without restriction under the CC0 license (Creative Commons Public Domain Dedication).
Where to download: The "Data Releases" page on the CIViC website provides direct access to the files. 
ClinVar
A public archive from the National Institutes of Health (NIH) that collects information on genomic variation and its relationship to human health. 
Data availability: You can download the full ClinVar dataset in various formats via their FTP site, including a compressed XML file and a tab-delimited summary file.
License: As an NIH resource, the data is publicly available for a wide range of uses.
Where to download: Downloads are available through the NCBI's FTP site (ftp.ncbi.nlm.nih.gov/pub/clinvar). 
Cancer Genome Interpreter (CGI)
This tool identifies tumor alterations that may be therapeutically actionable based on computational methods and curated databases. 
Data availability: Key datasets used by CGI, including "Cancer Biomarkers" and "Validated Oncogenic Mutations," are available for download in a zipped folder.
License: CGI's resources are freely available for non-commercial use.
Where to download: The "Datasets" section of the CGI website contains the downloadable files. 
OncoKB™
A precision oncology knowledge base from Memorial Sloan Kettering Cancer Center (MSK) that provides information on the effects of cancer gene alterations. 
Data availability: OncoKB does not support the bulk download of all annotated variants, recommending API access instead.
License: Access via API is subject to licensing terms. The public GitHub repository may offer some content, but not the full database.
Where to download: Bulk downloads are not an option. For researchers, API access is the primary method for annotation. 
Database of Curated Mutations (DoCM)
This is a small, focused database of functionally important cancer variants with manually curated citations. 
Data availability: DoCM is licensed under the Creative Commons Attribution license (CC BY 4.0), and its contents can be accessed via a web interface or an API.
License: CC BY 4.0 allows for free access for both academic and industry researchers.
Where to download: Access is provided through the web interface and API, not as a bulk download file.
